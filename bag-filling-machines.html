Bag Filling Machine

Bag Filling Machine

Adorn manufactures Open Mouth Bag Filling Machine in weighing capacities of 10 Kgs. to 50 Kgs. that are widely used in feed, chemical and food industries. These programmable bag filling machines with electronic load cell weighing has strong functionality and are easy to operate.

Type of Bag Filling Machine based on the material for filling

  • Powder Bag Filling Machine
  • Granule Bag Filling Machine

Features

  • Simple, User friendly
  • Microcontroller based / PLC based weighing electronics
  • Programmable Set Points for coarse, fine feed & in-flight
  • Provision for external batch start
  • Cumulative batch and quantity totalizer
  • Calibration through keyboard
  • Easy trouble shooting indications available in the controller
  • Pneumatically operated Bag Clamping arrangements & Control valves
  • Printer interface/PC Interface
